# Projeto de Listagem de Livros

## Tecnologia utilizadas
### Frontend
Reactjs + Vite
CSS module 
#### Dependencias
Styled Components, Axios, React-router-dom

### Backend Fake arquivo db.json
Nodejs
#### Dependencias
json-server

### Rodar o projeto Frontend
``` 
git clone
cd <nome-do-projeto>
npm install
npm run dev
```

### Rodar o projeto Backend Fake
```
cd api
npm install
npm start
```
